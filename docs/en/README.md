## Getting Started

1. **Installation**
   ```bash
   pip install -e .
   ```

2. **Generate Documentation**
   ```bash
   codeviewx -w /path/to/project -o docs
   ```

3. **Browse Documents**
   ```bash
   codeviewx --serve -o docs
   # Visit http://127.0.0.1:5000
   ```
